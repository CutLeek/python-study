加快pip安装速度

```
Linux下修改~/.pip/pip.conf文件，内容如下
[global]
index-url = http://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host=mirrors.aliyun.com

Windows下修改C:\Users\用户名\pip\pip.ini文件
[global]
index-url = http://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host=mirrors.aliyun.com
```
